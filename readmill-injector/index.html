Readmill Injector

Easily import books to Readmill from your favourite sites.

Readmill Injector plugin available as Chrome Extension or as a Firefox Add-On. Readmill Injector inserts 'Send to Readmill' buttons on Project Gutenberg and Feedbooks. This makes it really easy to send books to Readmills Library and thus, easy to send to their iPad application.

Or install from Chrome Webstore
Or install from Mozilla Add-On.

Source Code

Both the Chrome extension and the Firefox Add-on is available as open-source on Github.
